# 🌍 Language Detection App

An NLP-based Language Detection Web Application built using Python, Scikit-learn, and Streamlit.

The application can detect multiple languages from text input and voice input using Machine Learning and Speech Recognition.

---

# 🚀 Features

- Language Detection using NLP
- Text Input Support
- Voice Input Support
- Confidence Score Prediction
- Top Language Predictions
- Streamlit Web Interface

---

# 🛠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes
- SpeechRecognition
- Pandas

---

# 📂 Project Workflow

Text / Voice Input
↓
TF-IDF Vectorization
↓
Multinomial Naive Bayes Model
↓
Language Prediction
↓
Confidence Score Display
